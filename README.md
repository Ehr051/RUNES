# Nova Runas Vikingas

Landing page educativa para aprender a leer runas vikingas (Elder Futhark).

## Instrucciones

1. Abra el archivo `index.html` en un navegador para acceder a la página.
2. La página es responsiva y funciona en dispositivos móviles.
3. La página utiliza almacenamiento local (localStorage) para guardar las lecturas del diario.

## Estructura del proyecto

* `index.html`: Archivo principal de la página.
* `styles.css`: Archivo de estilos CSS para la página.
* `script.js`: Archivo de script JavaScript para la página.
* `runas.json`: Archivo JSON que contiene la información de las runas.
